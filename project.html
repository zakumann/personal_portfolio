클론 코딩-1

클론 코딩-1 페이지주소

Resoft 웹페이지 클론 코딩

기간 : 2022년 7월 1일 ~ 7월 15일 (총 15일)

담당 역할: 카카오 지도 API

내용: 사이트 도메인 활용 및 응용

성과 : 사이트 도메인 활용과 라이브러리 응용법 학습

여기서 맡은 부분은 카카오 API입니다. 팀은 각자 맡은 부분을 나눠서 진행했었습니다. 카카오 API는 처음에는 단순히 카카오 API 링크를 붙여놓는 것이라고 생각했었으나 사이트 도메인이 지도 API를 나타낼 사이트와 일치하지 않으면 지도가 제대로 나타나지 않았습니다. 이는 사이트 도메인이 지정한 위치에 나타나기 때문에 정확한 주소가 필요했습니다. 그 때 localhost를 임시 사이트 도메인으로 사용했었으며 이후, 과제로 제출할 때, 깃허브 페이지와 일치하게 바꿨습니다. 그리고, HTML의 특성상, 지도 API 밑에는 지도 주소 및 전화번호가 아래에서 나왔습니다. 이 문제를 해결하기 위해서 원본 사이트를 참조해서 지도 API와 글을 양 쪽 옆으로 나열시켰습니다. 이 배열은 WOW.js와 제이쿼리를 응용했습니다. 그 결과, 지도 API와 글은 이후 접속해서 해당 페이지를 내리면 슬며시 나타나는 애니메이션 효과가 적용되었습니다. 이를 통해 제이쿼리와 WOW.js와 같은 라이브러리 응용법을 배웠습니다.

클론 코딩-2

클론 코딩-2 페이지주소

기간 : 2022년 7월 18일 ~ 8월 23일 (총 36일)

담당 역할 : 홈페이지 제작 및 사업현황 관리

내용 : 코드 수정 및 프로젝트 운영 관리

성과: 시간 내 프로젝트 완료

두 번째로 맡은 팀 프로젝트는 사업현황을 나타내는 것으로 처음에는 PC판을 완성시켰다가 이후, 태블릿판과 모바일판의 제작을 위해서 팀 측에서 코드 수정을 요청한 적이 있었습니다. 처음에는 코드 수정을 일일이 했었다가 이내 코드를 처음부터 다시 새로 작성한 것이 상대적으로 쉽다는 판단에 이전에 작업했던 페이지를 처음부터 다시 작성한 다음에 따로 자체적으로 CSS를 적용시켰었습니다. 코드 수정을 마친 이후, 이번에는 서브페이지의 링크가 다른 팀원들과 맞추기 위해서 이후 제목의 CSS에 패딩 값을 주어서 팀원들과 위치를 맞췄습니다. 반응형을 적용할 때도 처음에는 공통 CSS로 갔었다가 반응형에 사소한 오류가 나오면 자체적으로 CSS를 추가해서 팀원들과 맞췃습니다. 이후로도 피드백이 들어오는 대로 수정을 해왔었고 정보마당에도 팀 간의 불화가 발생하지 않도록 허용하는 한도 내에서 수정을 했었습니다. 이 팀 프로젝트를 통해서 팀 간의 불화가 일어나지 않도록 주의를 기울여야했었고 팀 간의 약속을 지켰습니다.
